# Liguistic Fibbage with Flask-SocketIO

This is a multiplayer trivia game built with Flask and Socket.IO. Players can connect to the server, submit their answers to trivia questions, and see the results in real-time.

## Features

- **Real-time Gameplay:** Players can submit their answers and see the results instantly.
- **Multiplayer Support:** Multiple players can join the game and compete against each other.
- **Dynamic Question Pool:** Questions are randomly selected from a pool, ensuring each game is unique.
- **Score Tracking:** Players' scores are tracked throughout the game, and final scores are displayed at the end.
